# View名称
HealthChart
## 说明
> 仿苹果健康图表，图形边缘修改成非圆形。暂只支持柱状图

## 效果图

## 对外提供方法列表
>setAnimation(boolean);//设置是否动画显示<br>
>setChartType(HealthChart.ChartType);//设置图表类型，柱状图OR折线图<br>
>setData(HealthChart.HealthChartData);//设置数据<br>
